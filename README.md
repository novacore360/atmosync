# Atmosync - Premium Weather App 🌤️

A premium progressive weather web app with adaptive wellness goals, real-time awareness, and intelligent recommendations.

## ✨ Features

- 🌍 **Real-Time Geolocation Weather** - Automatic location detection with accurate weather data
- 🎨 **Dynamic Adaptive Themes** - UI changes based on time of day and weather conditions
- 🧠 **Smart AI Suggestions** - Personalized recommendations based on weather
- 💪 **Wellness Goals System** - Track hydration, sleep, exercise, and air quality
- 📱 **PWA Support** - Installable on all devices with offline functionality
- 🔔 **Smart Notifications** - Weather alerts, reminders, and emergency warnings
- 📊 **Statistics & Analytics** - Beautiful charts and weather history
- 🔄 **Cloud Sync** - Sync data across devices with Supabase
- 📡 **Offline-First** - Works without internet using IndexedDB

## 🚀 Quick Start

### Prerequisites

- Node.js 18+
- npm or yarn
- API keys from:
  - [OpenWeatherMap](https://openweathermap.org/api)
  - [WeatherAPI](https://www.weatherapi.com/)
  - [AirVisual](https://www.iqair.com/air-pollution-data-api)
  - [Supabase](https://supabase.com/)
